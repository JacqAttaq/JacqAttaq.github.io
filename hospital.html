Web application for managing facilities, patients, medications and doctors. Created using Rails, Bootstrap, and the Faker gem. Tested with Capybara.

  • Facility Index - Created hospital facilities with full CRUD capabilities.

  • Facility View - Indiviual facilities contain connections to all facility information, including patients and docotors

  • Facility Create - Form for creating and editing rendered from the same partial file to maintain DRY principles.

  • Pagination - Use of Will Paginate gem allowed for division of information with page breaks and numbering

  • Rake Task - Generated this task to create a large number of facilities and patients for research and testing.

  • Patient Index - Patients exist with full CRUD capabilities as a child of Facilities.

  • Patient Edit - Able to update all patient characteristics, including medications and status.

  • AJAX request - Changes "Sumbit" button to active when requirements met, without reloading page.
