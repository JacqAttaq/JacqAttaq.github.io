Web application for instructor/student management and communication. Built using Rails, Bootstrap and the Devise gem.

  • Course Index - Made index with full CRUD for courses. The courses have a relation to Locations and Users


  • Course View - Can view all locations where course is offered.


  • Location Create - Used partial


  • Location View - Can view all location information and course offerings on one page.


  • Location Edit - Can edit the Location, as well as the associated courses.


  • Devise Login - Installed the Devise gem in order to create login process and administrative permissions


  • Routes - Nested, polymorphic and relational sources were created for this application.
